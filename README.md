# 农作物种植决策优化（数学建模）

本仓库用于保存农作物种植决策优化项目的代码、数据与结果，便于复现与后续维护。

## 项目结构

- `支撑材料/支撑材料/问题1`：问题1脚本与数据
- `支撑材料/支撑材料/问题2`：问题2脚本与数据
- `支撑材料/支撑材料/问题3`：问题3脚本、模拟退火算法与结果
- `C2024021001076.pdf`：说明文档

## 运行建议顺序

1. 进入 `问题1` 运行 `trouble1_1.m`、`trouble1_2.m`
2. 进入 `问题2` 运行 `mengtekaluo2.m`
3. 进入 `问题3/模拟退火算法` 运行 `mo_ni_tui_huo.m`
4. 按需运行 `问题3` 下分析脚本（如 `chanliang_fenbu.m`）
